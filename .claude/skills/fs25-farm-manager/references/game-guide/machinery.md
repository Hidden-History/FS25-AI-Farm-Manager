# Vehicles, Machines & Tools

**Sources:** `FS25HF-manual_EN.pdf` ("Highlands Fishing" edition, 27pp) — **primary**, pp.22-23
("Vehicles, Machines & Tools"). Cross-checked against `FS25-manual_EN.pdf` (base edition, 24pp),
same pp.22-23: **programmatically confirmed byte-identical.** Academy "Machinery 101" content
previously in this file has been **re-verified against both PDFs** per the corrected sourcing
policy (the FS25 Academy website carries unrevised Farming Simulator 22 content and is no longer
a primary source) — neither PDF corroborates any of it (searched directly for loader-type names:
Forklift, Skid-Steer, Telehandler, Wheel Loader — all absent).

**This file is a distillation, not the live source of truth** — see the in-game Help for the
live version of this content.

## AI Workers & GPS-Assist (FS25HF p.22, base p.22 agrees)

- **Hiring a worker:** press the helper button to get AI assistance with tasks — plow, sow,
  harvest are the manual's given examples. The worker "expects a wage," auto-deducted from your
  bank account. (No wage amount/formula given.)
- **GPS-assisted steering:** can either steer for the player directly, or make a hired AI worker
  more precise. The system creates individual "ideal lines" per field.

## Machine Dealership — Buying & Leasing (FS25HF p.22, base p.22 agrees)

- Equipment is browsable by category or brand at the dealership.
- Suggested combinations (matching tool ↔ vehicle) can be selected with a single button press
  when offered.
- **Leasing** is available as an alternative to buying: a starting fee plus an ongoing leasing
  fee (no numbers given).
- **Used vehicle sale:** discounted listings worth checking, per the manual.

## Vehicle Configurations (FS25HF p.23, base p.23 agrees)

- Some vehicles support configuration changes: wheels, engine power, color, or attachments like a
  front loader.
- Configure at purchase time in the shop, or later by bringing the vehicle to the marked area in
  front of the machinery dealer and interacting with the configuration symbol.

## Vehicle & Machine Repair (FS25HF p.23, base p.23 agrees)

- Vehicles/tools deteriorate over time; a vehicle needing repair loses speed or engine power.
- Repair at the machinery dealer or your own vehicle workshop (both cost money; no pricing
  given).
- Paintwork wear is cosmetic and fixed separately by repainting.

## Shop Icons glossary (FS25HF p.23, base p.23 agrees)

| Icon | Meaning |
|---|---|
| Power | Vehicle's minimum engine power |
| Required Power | Engine power required to properly operate the tool |
| Transmission | Vehicle's transmission type |
| Speed | Vehicle's maximum achievable speed |
| Working Speed | Maximum speed while actively working with the vehicle/tool |
| Fuel | Fuel tank capacity or electric charge capacity |
| Weight | How heavy the vehicle/tool is |
| Max Fill Weight | Recommended maximum fill weight for the tool |
| Width | Working width of the vehicle/tool |
| Fill Type | What the vehicle/tool can transport or process |
| Bales | Type/size of bales the tool can transport or create |
| Seeds | Type of crops the tool can sow |
| Animals | Type and quantity of animals a trailer can transport |

## Academy-sourced, NOT in either FS25 manual — unverified, and one item dropped entirely

A loader-type decision reference (Forklift → pallets/bigbags; Front Loader → all-rounder;
Wheel Loader → heavy material/forestry; Skid-Steer Loader → narrow spaces; Telehandler → high
places) was previously sourced to Academy news_id=412/386 ("Machinery 101," its only 2
sub-articles). Searched both PDFs directly for every loader-type name (Forklift, Skid-Steer,
Telehandler, Wheel Loader) — all absent from both editions. Kept here only as a plausible,
unverified pointer: five loader types likely exist with roughly these use cases, but none of
this is manual-confirmed.

**Dropped entirely, not carried forward even as unverified:** both of those same two Academy
articles also contained the identical sentence *"In Farming Simulator 22, [tractors of the]
John Deere 7R Series [are] only compatible with the frontloader 700M"* — this is one of the
**confirmed FS22-tainted** instances (see `index.md`'s punch list), so unlike the loader-type
list above, this specific claim isn't even kept as an unverified note.
