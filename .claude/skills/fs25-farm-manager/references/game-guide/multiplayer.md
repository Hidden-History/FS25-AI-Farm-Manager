## Multiplayer (FS25HF p.27, base p.24 agrees)

- Farms can be run together, or players can manage individual farms while helping each other.
- Requires an internet connection; in-session chat via a chat button; "most systems support
  cross-platform multiplayer."
- **Join game:** browse currently running sessions; if the host is running mods, joining
  requires installing the same mods (listed in the game's details).
- **Create game:** choose a save game, difficulty, and map (an existing career save can be
  reused); additional settings include max player count and mod list.
- **Multiplayer options:** the host can grant fellow players permissions — e.g. to buy/sell
  machines or fields, or start contracts.
- **Banning players:** the server admin can ban a misbehaving player via the players list.
- Further help: `farming-simulator.com/support`.

## Modifications (Mods) (FS25HF p.27, base p.24 agrees)

- Community-created extensions adding machines, maps, or gameplay mechanics.
- Official, curated source: the **ModHub** ("tested and curated by GIANTS Software").
- **Adding mods:** main menu → "Downloadable Content" → browse ModHub categories → install by
  clicking (auto-downloads and installs).
- **Creating mods:** documentation/tutorials/forum at `gdn.giants-software.com`.

## Farming Simulator Academy (FS25HF p.27, base p.24 agrees) — manual's own closing pointer

The manual's last page points players to the in-game help section and to
`farming-simulator.com/academy` for further tutorials — the manual itself names the Academy as a
companion resource. **Updated understanding:** this guide no longer treats the Academy website as
a primary source (it was found to carry unrevised Farming Simulator 22 content — see `index.md`'s
punch list); the in-game help section the manual also points to is the one still treated as
authoritative here (see this guide's own top note).
