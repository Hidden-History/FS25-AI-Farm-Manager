# Step 3: Save it twice, deliberately

**Progress: Step 3 of 8** — Next: Check the skill's own honesty

**Save it twice, deliberately**: archive to `sanctum/history/journal/{{date}}-session-{{n}}.md`,
then overwrite `sanctum/history/closeout-latest.md` with the same content. The archive is history;
`history/closeout-latest.md` is what next session actually reads. These are two different jobs —
writing only one silently breaks either the record or the handoff.
